# ECommerceSystemBl
# ECommerce System - Blazor

A full-stack E-Commerce web application built with **Blazor Web App (.NET 10)** as the frontend and **ASP.NET Core Web API** as the backend. This project was developed as part of my Full Stack .NET training to apply modern web development concepts using the .NET ecosystem.

---

# About

The application simulates a real-world online shopping platform where customers can browse products, manage their shopping cart, place orders, and view their order history. It also includes an Admin Dashboard for managing products, categories, users, and customer orders.

The frontend is built with **Blazor Web App**, while the backend communicates through RESTful APIs and uses **Entity Framework Core** with **SQL Server** for data management.
